---
description: Create a new SaaS explainer video using the Reference System
---

## 1. Preparation
Ask the user to fill out the `workflow_input_requirements.md` template found in `src/doc/video-system/`.

## 2. Analysis
Read the following files to understand the system:
1. `src/doc/video-system/system_prompt_rules.md` (Your core instructions)
2. `src/doc/video-system/saas_video_guidelines.md` (The style guide)
3. `src/doc/video-system/component_library.md` (The Lego bricks)
4. `src/doc/video-system/scene_library.md` (The reference scenes)
5. `src/doc/video-system/creating_new_scenes_guide.md` (How to innovate)

## 3. Planning
Based on the user's input:
- Plan the scene sequence.
- Decide which existing scenes to reuse.
- Decide which new scenes need to be created (and how they will meet the quality guidelines).

## 4. Execution
Create the product directory `src/products/[ProductName_VX]` and implement the video.

# Workflow Input Requirements: Raw Product Knowledge

**Goal**: Provide enough context for the AI to generate a high-quality script and video. **Do NOT write the script yourself.** Provide the raw facts, features, and benefits.

## 1. Product Identity
*   **Name**: (e.g., "ExpertFlow")
*   **One-Line Pitch**: (e.g., "The smartest way to connect with experts.")
*   **Logo**: SVG or high-res PNG (White/Transparent preferred).
*   **Brand Color**: Hex code (Primary accent color).

## 2. Target Audience & Pain Points
*   **Who is this for?** (e.g., "HR Managers", "DevOps Engineers")
*   **Top 3 Frustrations**: What are they complaining about right now?
    *   *Example*: "Too many emails", "Lost in spreadsheets", "Waiting for approvals".
*   **The "Enemy"**: What is the old way of doing things? (e.g., "Excel", "Endless Meetings").

## 3. Product Solution
*   **The Big Promise**: How do you solve the pain in 3 words? (e.g., "Instant Expert Access").
*   **Key Features (The "How")**:
    *   *Feature 1*: (e.g., "AI Smart Matching - Connects you to the right expert instantly").
    *   *Feature 2*: (e.g., "Knowledge Base - Automatically saves answers for later").
    *   *Feature 3*: (e.g., "Mobile App - Ask from anywhere").

## 4. Visual Assets (Optional but Recommended)
*   **UI Screenshots/Figma**: Links or file paths to key screens.
*   **Wallpapers**: Abstract backgrounds to use on device screens.
